# Avrora Atelier — пример лендинга

## Ссылка на GitHub pages
https://z1ec.github.io/RDN-example-avrora-2 

Этот репозиторий — демонстрационный лендинг, который я сделал как пример своих работ. Проект показывает стиль, типографику, композицию блоков и базовую интерактивность.

## Что внутри
- Главная страница с несколькими секциями (hero, коллекция, информация, видео, футер).
- Минимальная интерактивность на чистом JavaScript (уведомление при кликах).
- Кастомные стили на Tailwind CSS.

## Технологии
- HTML5
- Tailwind CSS
- JavaScript (vanilla)
- npm scripts для сборки CSS

## Запуск локально
1. Установить зависимости:
   `npm install`
2. Собрать CSS:
   `npm run build:css`
3. Открыть `index.html` в браузере.

Для разработки со слежением за изменениями:
`npm run watch:css`

## Примечание
Проект предназначен для портфолио и демонстрации подхода к верстке, визуальному стилю и структуре страницы.
